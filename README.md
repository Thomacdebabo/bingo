# Prediction Bingo

Website which hosts prediction bingos with your friends. The idea is that everybody will have predictions for the next year and we just make a bingo out of that. Of course this should also be able to do adaptable to other events where you want a bingo.

# Overview

## Components
- Frontend
- Backend
  - processing
  - data storage

## Functionality

- Adding predictions per user
- aggregate predictions over users
- combine predictions to create bingo cards
- verify predictions
- update the cards

## Data
- User
  - username, mail, password (usermanagement)
  - predictions list
- Prediction
  - name, category, description, id, image, verification
- Bingo Card
  - 2d predictions array
